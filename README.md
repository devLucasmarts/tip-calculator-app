## How to download and install ⚙️🖥️
<br />

#### git clone 🔽

<pre>
git clone git@github.com:martsDev/tip-calculator-app.git
</pre>

<br />

### Access tip-calculator-app directory 📂

<pre>
cd tip-calculator-app
</pre>

<br />

### Install the dependencies 🔧

<pre>
npm install
</pre>

<br />

### Run the project! ▶️

<pre>
npm start
</pre>
